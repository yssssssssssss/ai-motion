# Motion Knowledge

MotionLens 的动效知识库基础包，当前用于把设计规范和场景案例转成可检索的结构化知识，供 AI 分析稿件时作为上下文参考。

## 来源

- `动效设计规范系统（ing~）.pdf`：沉淀原则、参数范围、风险约束和通用动效模式。
- `人文共情-创造更有温度的用户体验-动态化趣味性场景设计预研.pdf`：沉淀电商链路场景、触点案例、情绪价值和趣味化边界。

## 内容类型

- `principle`：设计原则和约束。
- `pattern`：可推荐的动效模式与参数范围。
- `case`：购物前、购物中、购物后的业务场景案例。
- `risk-rule`：不应推荐或需要降级的风险判断。

## 生产验收

- `validateMotionKnowledgeBase()`：检查重复 id、缺失来源、未知来源、无页码、无摘录、参数范围错误等问题。
- `createKnowledgeProductionManifest()`：输出来源数量、条目数量、类型覆盖、来源覆盖和校验问题，可作为后续知识库更新后的验收摘要。

## 当前边界

这个包是 P0 形态，不包含向量数据库、后台管理、在线文档解析或自动引用生成。检索使用确定性打分，便于测试和调试；后续可以在不改调用方契约的前提下替换为 RAG 检索服务。
