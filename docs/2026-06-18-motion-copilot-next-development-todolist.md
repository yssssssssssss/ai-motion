# Motion Copilot 下一阶段开发 Todo

日期：2026-06-18
更新：2026-06-23

范围：仅限独立模块 `apps/motion-copilot` 与 `packages/motion-copilot-core`，不影响项目既有功能。

当前状态：上一阶段功能项已基本完成。当前模块以“当前图层 / 图层轨道 / 多图层时间轴”为核心语义，不再暴露“目标对象 / 主目标 / 选中图层”切换概念；编排片段以“开始时间 + 时长”表达，内部保留 `delayMs` 仅用于兼容和串并行换算。

  - 验收：支持上移、下移、置顶、置底。

## P1：多图层时间轴增强

- [x] 增加时间刻度。
  - 验收：用户能看清每个动效片段的开始时间、结束时间和持续时间。
  - 已完成：编排区顶部展示时间刻度，动效卡片内展示起止时间窗，右侧面板可编辑选中片段开始时间。
- [x] 支持动效卡片拖动到具体时间位置。
  - 验收：水平拖动改变开始时间。
  - 验收：垂直拖动到不同轨道时改变作用图层。
  - 已完成：同轨水平拖动会更新片段开始时间，跨轨拖动会改变作用图层并保留落点时间。
- [x] 增加吸附与对齐能力。
  - 验收：不同轨道卡片上下对齐表示并行。
  - 验收：卡片靠近其他片段起点 / 终点时出现吸附线。
  - 已完成：拖动时靠近其他片段起点 / 终点会吸附，并显示落点线与吸附线；上下对齐后的片段按同一开始时间并行播放。
- [x] 支持轨道级操作。
  - 验收：轨道可折叠 / 展开。
  - 验收：轨道头显示图层名、显隐、锁定状态。
  - 已完成：轨道头显示显隐/锁定摘要，轨道可折叠展开，折叠后保留轨道状态与片段计数。
- [x] 右侧面板支持片段专属参数。
  - 验收：选中不同动效片段时，只展示该动效可调整的参数。
  - 验收：修改片段参数后，预览和时间轴立即同步。
  - 已完成：选中动效片段后，可编辑开始时间、时长、串并行、缓动，以及起止 X/Y/Scale/Opacity/Blur。
  - 已完成：底部动效卡片展示“开始时间 + 时长”，不再把内部延迟字段作为用户高频参数暴露。

## P2：规范动效按钮体系

- [x] 继续从动效规范中提炼明确命名的动效。
  - 验收：按进场、退场、位移、反馈、遮罩、转场、加载、手势等类别组织。
  - 已完成：当前规范库已沉淀 21 个明确命名动效，并按推荐、组合、转场、导航、横向、弹窗、底板、遮罩、卡片、反馈、加载、骨架、列表等分类组织。
- [x] 将规范动效拆成可组合的原子按钮。
  - 验收：单个按钮可一键应用到当前图层，或添加到多图层编排轨道。
  - 验收：多个按钮组合时能生成新的组合动效。
  - 已完成：每个规范动效都可单独应用或添加到编排轨道，组合预设通过多个原子动效生成。
- [x] 增加规范约束。
  - 验收：每个按钮带默认时长、缓动、方向、适用层级、是否允许回弹。
  - 验收：冲突组合能给出提示或自动修正。
  - 已完成：按钮和规范库展示槽位、默认时长、弹性/非弹性信息；骨架屏、同类槽位、当前图层不可用等冲突会提示或自动修正。
- [x] 增加组合预设。
  - 验收：常见 App 场景可一键生成多图层组合，例如弹窗进场、底部面板、Tab 切换、加载转成功。
  - 已完成：新增弹窗进场、底部面板、Tab 切换、加载转成功四类组合按钮。
- [x] 优化动效库信息架构。
  - 验收：左侧动效库不再纵向全量展开，避免模块臃肿。
  - 已完成：增加分类 Tab、搜索、当前图层推荐；横向分类条隐藏浏览器滚动条但保留横滑能力。

## 下一阶段建议执行顺序

1. 将 `apps/motion-copilot` 与 `packages/motion-copilot-core` 正式纳入版本管理，避免独立模块停留在未跟踪目录。
2. 准备发布演示脚本：新建项目、上传图层、选择规范动效、多轨道编排、调开始时间/时长、循环预览、导出。
3. 增强导出能力：补充 JSON 编排协议、研发可读参数表、React/CSS 片段。
